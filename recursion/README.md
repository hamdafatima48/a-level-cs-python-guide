# Recursion

This folder contains Python code related to recursion for A-Level Computer Science Paper 4.

The first file in this section is a general recursion example. It explains the concept in detail, including what recursion is, why a base case is needed, how the recursive call works, and how the function eventually stops.

After that, I have included Paper 4 recursion questions. In Paper 4, recursion often appears in two main ways:

* an iterative function is given, and the task is to write the recursive version
* a recursive function has to be written directly from the question

## What This Section Focuses On

* Understanding what recursion means
* Base cases
* Recursive calls
* How recursive functions stop
* Comparing iterative and recursive solutions
* Paper 4 recursion-style questions

## How to Use

Start with the general recursion example first. It is heavily commented to explain what is happening at each step.

Then move on to the Paper 4 questions to see how recursion is used in actual exam-style problems.
