# Data Structures

This folder contains Python code related to data structures for A-Level Computer Science Paper 4.

Data structures are an important part of Paper 4 because they are used to store and organise data in different ways. Some structures are simple, like arrays, while others need more careful understanding, like linked lists, stacks, queues, and binary search trees.


## What This Section Focuses On

* Linked lists
* Stacks
* Queues
* Binary search trees
* Arrays/lists used as data structures
* Adding, deleting, searching, and updating data
* Understanding pointers/indexes
* Paper 4 data structure questions

## How to Use

Start with the basic linked list example first, because linked lists can feel confusing when you first study them.

The comments in the code are meant to explain the parts that are usually tricky, especially how pointers/indexes change when data is added or deleted.
