# Object-Oriented Programming

This folder contains my commented Python solutions to selected Cambridge A-Level Computer Science Paper 4 questions that focus mainly on object-oriented programming.

These are not short isolated examples. They are full solutions to longer Paper 4 questions that I practised while studying A-Level Computer Science. I have included comments in the code to explain what is happening at different steps, so I hope they can help other students understand how similar questions can be approached and solved.

## What This Section Includes

The solutions in this folder may involve:

* Classes and objects
* Attributes and methods
* Constructors
* Arrays/lists of objects
* Inheritance
* Class-based problem solving
* Writing longer structured programs for Paper 4

## Paper References

Each file includes a short reference to the paper, year, and question number for context.

The official question text is not copied or reproduced in this repository. Only my Python solutions and comments are included.

## Note

These solutions are shared as study support rather than official mark-scheme answers. They are meant to help with understanding the logic and structure behind Paper 4 programming questions.
