# File Handling

This folder contains Python code related to file handling for A-Level Computer Science Paper 4.

File handling is very common in Paper 4 because many questions use text files to store data. The code usually reads data from a file, processes it, stores it in arrays/objects, or writes updated data back to a file.

## What This Section Focuses On

* Opening and reading text files
* Reading data line by line
* Using `.strip()` to remove extra newline characters
* Splitting file data using `.split()`
* Converting file data into the correct data type
* Storing file data in arrays/lists
* Handling missing files with `try` and `except`
* Writing data back to a file, if needed

## How to Use

The comments in the code explain the parts that are usually confusing in Paper 4, especially how the file data is read and how it is stored.

For most questions, the important thing is understanding the pattern: open the file, read the data correctly, convert it if needed, store it, and close the file.
